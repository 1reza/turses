turses: a Twitter client with a sexy curses interface
=====================================================

``turses`` is a Twitter client for the console. 

Features
--------

- Multiple timelines (buffers)
- Multi-column 
- Tweet, Reply, Retweet, Delete tweet
- Follow/Unfollow
- Favorite/Unfavorite
- Direct Messages
- Open URLs in browser
- Thread view
- Unread count
- Search
- View any user's tweets
- Fully customizable
- Multiple accounts
