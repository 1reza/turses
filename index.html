turses

A Twitter client for the console.

Installation

From source

$ git clone https://alejandrogomez@github.com/alejandrogomez/turses.git
$ cd turses
$ python setup.py install

Python Package Index (PyPI)

$ pip install turses

or

$ easy_install turses

AUR

A package in the Arch User Repository (AUR) is available for Arch Linux users. You can install it with yaourt.

yaourt -S turses
